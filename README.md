## 🌐 Overview

This repository contains the source code for the **S360 Global** official web presence. 
Operating across Brazil and Europe for over three decades, S360 Global represents a strategic expansion into the United States—an investment in people, processes, and local capabilities across infrastructure, engineering, energy, and technology.

The interface was strictly designed to reflect an **ultra-premium, minimalist, and corporate** aesthetic (Top-tier Enterprise equivalent).

## 🚀 Tech Stack

- **Core:** React 18 + TypeScript + Vite
- **Styling:** Tailwind CSS v4
- **Animations:** Framer Motion
- **Iconography:** Phosphor Icons
- **Typography:** Copasetic Pro *(Headings)* & Figtree *(Body)*

## 🛠️ Getting Started

To run the project locally:

1. **Install dependencies:**
   ```bash
   npm install
   ```
2. **Start the development server:**
   ```bash
   npm run dev
   ```
3. **Build for production:**
   ```bash
   npm run build
   ```
